Understanding Linear Regression with Python

Linear regression is a fundamental statistical modeling technique used to understand and analyze the relationship between a dependent variable and one or more independent variables. It assumes a linear relationship between those variables, aiming to find the best-fit line representing this relationship. In this blog post, we will explore the fundamental concepts of linear regression, including its assumptions and evaluation metrics. Understanding these aspects is crucial for building reliable and accurate regression models. We will also learn to implement linear regression using Python through a practical example.

Basic of Linear Regression

The basic idea behind linear regression is to estimate the regression coefficients that minimize the sum of squared differences between the predicted values and the actual values of the dependent variable. The predicted values are calculated using a linear equation that combines the regression coefficients and the independent variables.

The equation for a simple linear regression with one independent variable is:

$$ y = \beta_0 + \beta_1 X + ε $$

Here, y represents the dependent variable, x represents the independent variable, β₀ represents the y-intercept (the value of y when x is 0), β₁ represents the slope (the change in y for a one-unit change in x), and ɛ represents the error term (the difference between the predicted and actual values of y that cannot be explained by the model).

In multiple linear regression, where there are more than one independent variable, the equation becomes:

$$ y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \beta_n X_n + ε $$

Here, x₁, x₂, …, xₙ represent the different independent variables, and β₁, β₂, …, βₙ represent the corresponding regression coefficients.

Evaluation of Linear Regression

Evaluating the performance of a linear regression model is essential to understand its accuracy and reliability in predicting the dependent variable. It involves assessing how well the model fits the data and quantifying the prediction errors. Several evaluation metrics are commonly used to evaluate linear regression models. Let’s discuss some of the key evaluation metrics:

Coefficient of Determination (R²): The coefficient of determination, denoted as R², measures the proportion of the variance in the dependent variable that can be explained by the independent variables. It ranges from 0 to 1, where 0 indicates that the model does not explain any variance, and 1 indicates a perfect fit. In other words, R² represents the goodness of fit of the model. Higher R² values indicate a better fit, suggesting that a larger portion of the variance in the dependent variable is accounted for by the independent variables.

$$ R^2 = 1 - {RSS \over TSS} $$

Mean Squared Error (MSE) and Root Mean Squared Error (RMSE): MSE calculates the average of the squared differences between the predicted and actual values. It measures the overall prediction error of the model. RMSE is the square root of MSE and provides a more interpretable metric in the original units of the dependent variable. Both MSE and RMSE provide a measure of how close the predicted values are to the actual values. Lower values of MSE and RMSE indicate better model performance, as they reflect smaller prediction errors.

$$ MSE = {1 \over n} \sum_{i=1}^n (y_i - \hat {y_i})^2 $$

$$ RMSE = \sqrt{MSE} $$

Adjusted R²: While R² provides a measure of the overall goodness of fit, it may increase even with the addition of irrelevant variables to the model. Adjusted R² addresses this concern by penalizing the addition of unnecessary variables. It takes into account the number of independent variables and adjusts the R² value accordingly. Adjusted R² is often used to compare models with different numbers of variables and to select the model with the best balance of simplicity and explanatory power.

$$ Adjusted \ R^2 = 1 - {(1-R^2)(n-1) \over n-p-1} $$

Assumptions of Linear Regression

Linear regression relies on several key assumptions to ensure the validity and reliability of the model’s results. Let’s dive deeper into each assumption:

Linearity: The relationship between the independent variables and the dependent variable should be linear. This means that the change in the dependent variable is directly proportional to the change in the independent variables. One way to assess linearity is by examining scatter plots of the independent variables against the dependent variable. If the points exhibit a clear linear pattern, the linearity assumption is likely satisfied. In case of non-linear relationships, transformations or polynomial terms may be used to capture the non-linearity. Scatterplots illustrate the relationship between the independent variables and the dependent variable. The left scatterplot shows a clear linear pattern, indicating a strong linear relationship. In contrast, the scatterplot on the right does not exhibit a linear pattern, suggesting a non-linear relationship between the variables.

Independence of Errors: The errors or residuals (the differences between the predicted and actual values) should be independent of each other. This assumption assumes that the residuals do not exhibit any patterns or correlations. To assess this assumption, a plot of residuals against any relevant variable should not reveal any visible patterns. The left plot displays random scatter, indicating unbiased predictions, while the right plot shows a clear pattern in the residuals.

Constant Variance of Errors (Homoscedasticity): The variance of the errors should be constant across different levels of the independent variables. In other words, the spread of the residuals should not systematically change as the values of the independent variables change. A plot of residuals against the predicted values can help identify if there is a violation of this assumption. Ideally, the plot should exhibit a random scattering of points around the horizontal line.

Normality of Errors: The errors should follow a normal distribution. This assumption is crucial for hypothesis testing, confidence intervals, and calculating prediction intervals. We can assess the normality assumption by examining a histogram or a Q-Q plot of the residuals. If the residuals approximately follow a bell-shaped curve, the normality assumption is likely satisfied. The histograms provide a visual representation of the distribution of errors, while the Q-Q plots compare the quantiles of the residuals against a theoretical normal distribution.

No Multicollinearity: In multiple linear regression, the independent variables should not be highly correlated with each other. Multicollinearity can lead to unstable coefficient estimates and make it challenging to interpret the individual effects of the independent variables. One way to assess multicollinearity is by calculating the correlation matrix between the independent variables. Correlation values close to -1 or 1 indicate a high degree of collinearity, while values close to 0 indicate no or weak correlation. The left heatmap displays a high correlation between variables, indicating multicollinearity. The right heatmap shows no significant correlations among variables, indicating no multicollinearity.

Linear Regression with Python

In this example, we start by loading the “tips” dataset from Seaborn. We then select the independent variables (total_bill and size) and the dependent variable (tip).

import seaborn as sns
import pandas as pd
import numpy as np
import statsmodels.api as sm
from sklearn.linear_model import LinearRegression
from sklearn.metrics import r2_score, mean_squared_error
from scipy import stats

# Load the tips dataset from Seaborn
tips = sns.load_dataset('tips')

# Select the independent variables (X) and the dependent variable (y)
X = tips[['total_bill', 'size']]
y = tips['tip']

Next, we add a constant term to the independent variables matrix using sm.add_constant(X). This is required by the statsmodels library for calculating the intercept term in the linear regression model.

# Add a constant term to the independent variables matrix
X = sm.add_constant(X)

We fit the linear regression model using sm.OLS(y, X).fit(). The OLS function stands for Ordinary Least Squares, which is the method used for estimating the model parameters.

We print the summary of the model using model.summary(). This provides a comprehensive overview of the model’s coefficients, significance levels, goodness of fit statistics, and more.

# Fit the linear regression model
model = sm.OLS(y, X).fit()

# Print the model summary
print(model.summary())

To evaluate the model, we calculate the R-squared (r2_score), mean squared error (mean_squared_error), and root mean squared error (np.sqrt(mse)) using the predicted values and the true values.

We also calculate the adjusted R-squared (adj_r2) using the formula provided earlier. It adjusts the R-squared value based on the number of independent variables in the model.

# Evaluate the model
r2 = r2_score(y, y_pred)
mse = mean_squared_error(y, y_pred)
rmse = np.sqrt(mse)

# Calculate the adjusted R-squared
n = len(y)
p = X.shape[1] - 1
adj_r2 = 1 - (1 - r2) * (n - 1) / (n - p - 1)

# Print the evaluation metrics
print("Evaluation Metrics:")
print("R-squared (R²):", r2)
print("Mean Squared Error (MSE):", mse)
print("Root Mean Squared Error (RMSE):", rmse)
print("Adjusted R-squared:", adj_r2)

Conclusion

In this blog post, we explored the concept of linear regression and its application using Python. We started by understanding the basic idea behind linear regression and the equations for simple and multiple linear regression.

To evaluate the performance of a linear regression model, we discussed several key evaluation metrics. The coefficient of determination (R²) measures the proportion of the variance in the dependent variable that can be explained by the independent variables. Mean squared error (MSE) and root mean squared error (RMSE) quantify the overall prediction error of the model. Adjusted R² adjusts the R² value based on the number of independent variables to find the best balance of simplicity and explanatory power.

Linear regression relies on certain assumptions to ensure the validity and reliability of the model’s results. We explored the assumptions of linearity, independence of errors, constant variance of errors (homoscedasticity), normality of errors, and no multicollinearity. To assess these assumptions, we examined scatter plots, residuals plots, QQ plots, and correlation matrices.

We then implemented linear regression using Python with the help of the statsmodels and scikit-learn libraries. We loaded the “tips” dataset from Seaborn, prepared the independent variables and dependent variable, fitted the linear regression model, and evaluated its performance using R², MSE, RMSE, and adjusted R².
